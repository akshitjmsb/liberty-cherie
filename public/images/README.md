# Image Assets

Add your product and marketing images here.

## Required Images

### Hero & Marketing
- `hero-products.jpg` - Main hero image (recommended: 1200x1200px or larger)
- `about-maker.jpg` - Photo for about page (recommended: 800x1000px)

### Open Graph & Social
- `og-image.jpg` - Social sharing image (must be 1200x630px)

### Products
Place product images in `/products/` folder:
- Use descriptive names like `daisy-pouch-1.jpg`, `victoria-tote-2.jpg`
- Recommended size: 800x800px minimum, square format
- Support multiple images per product (e.g., `-1.jpg`, `-2.jpg`)

## Image Guidelines

1. **Format**: Use JPG for photos, PNG for graphics with transparency
2. **Quality**: Optimize images for web (use tools like TinyPNG or Squoosh)
3. **Naming**: Use lowercase, hyphens instead of spaces
4. **Aspect Ratio**: Product images should be square (1:1) for consistency

## Placeholder Images

During development, you can use placeholder services:
- https://placehold.co/800x800/faf8f5/c26e7a?text=Product
- https://unsplash.com (free stock photos)

Or update `next.config.ts` to allow external image domains.
