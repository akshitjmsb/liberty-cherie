# PWA Icons

Generate PWA icons using one of these tools:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Required Icons

From your logo/brand icon (recommended: start with a 512x512 or larger PNG):

| Filename | Size | Purpose |
|----------|------|---------|
| icon-72x72.png | 72x72 | Android icon |
| icon-96x96.png | 96x96 | Android icon |
| icon-128x128.png | 128x128 | Android icon |
| icon-144x144.png | 144x144 | Android splash |
| icon-152x152.png | 152x152 | iOS icon |
| icon-192x192.png | 192x192 | Android icon (required) |
| icon-384x384.png | 384x384 | Android splash |
| icon-512x512.png | 512x512 | Android icon (required) |

## Shortcut Icons (Optional)

| Filename | Size | Purpose |
|----------|------|---------|
| shop-icon.png | 96x96 | PWA shortcut icon |
| custom-icon.png | 96x96 | PWA shortcut icon |

## Apple Touch Icon

Also place in `/public/`:
- `apple-touch-icon.png` (180x180px recommended)

## Favicon

- `favicon.ico` - Standard favicon (already exists in /src/app/)

## Brand Colors

Use these brand colors for icon backgrounds:
- Primary: #c26e7a (Dusty Rose)
- Secondary: #7a9e7e (Sage Green)
- Accent: #d4a574 (Warm Gold)
- Cream: #faf8f5
